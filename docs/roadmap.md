# Roadmap

## MVP

- Local CLI.
- VS Code sidebar.
- Deterministic git diff risk scoring.
- Protected file warnings.
- Task scope locking.
- Context and task markdown files.

## Next

- GitHub Action wrapper.
- Bundled VS Code extension build.
- Better task metadata and task selection.
- Inline VS Code decorations for protected files.
- Configurable risk weights.
- Monorepo-aware package detection.

## Later

- Team policy sync.
- Hosted audit log.
- Pull request comments.
- Organization-level dashboards.
- SaaS billing and seats.
