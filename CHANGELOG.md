# Changelog

## 0.1.0

- Added local CLI with `init`, `scan`, `update-context`, `check`, `explain`, `revert-unrelated`, and task commands.
- Added deterministic risk scoring for protected paths, sensitive areas, dependency/config changes, deletions, broad edits, and unrelated changes.
- Added `AGENT_CONTEXT.md`, `AGENT_TASKS.md`, and `agentshield.config.json` generation.
- Added minimal VS Code sidebar.
- Added unit tests for config loading, risk scoring, protected path matching, unrelated change detection, and task overlap detection.
